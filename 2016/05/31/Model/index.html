Android项目架构-Model类编写

Model基础知识

Model包主要用于缓存数据,将数据保存在内部存储中。
涉及文件操作 File

实例 - Model类的编写

思路

  • 由于Model类是需要被继承的,所以要定义成abstract抽象类。
  • 由于抽象类不能被初始化,所以要在抽象定义一个内部类DataCache。
  • 在内部类定义通用的读写方法。
  • 封装读写方法,声明读写普通对象的方法以及读写集合的方法。

  • 读取方法的实现
