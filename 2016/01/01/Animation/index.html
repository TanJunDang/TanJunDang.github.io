Android动画

  • Android动画

基础知识

动画分帧动画(Frame)、补间动画(Tween)、属性动画(property)三大类。
使用
使用方式分java、xml两种。java只需创建相应的XXXAnimatino然后调用start方法即可。这里讲xml使用方式。
1.在res创建相应的目录.
2.创建相应的根标签的xml文件.
3.调用AnimationUtils(AnimatorInflater)加载xml文件。然后调用start方法即可。

动画类型 在res创建的目录 对应的标签
帧动画 drawable(不需创建) animation-list
补间动画 anim translate、alpha、rotate、scale
属性动画 animator animator、objectanimator、set

动画坐标图

帧动画(Frame)

每一帧是一张图片，通过在短时间内加载不同的图片形成动画效果。
需要在drawable目录下创建根标签为animation-list的文件。

帧集合(animation-list)

使用
1.在drawable目录创建根标签为animation-list的xml文件
2.添加子标签Item，为每个item设置drawable以及duration
3.XML布局中ImageView的background属性使用这个文件
4.创建AnimationDrawable引用，通过ImageView的getBackground方法来获取对象。
5.调用AnimationDrawable对象的start、stop方法即可。

补间动画(Tween)

理解
只能对View实现旋转、平移、缩放、透明效果，无扩展性。
只是重新绘制Drawable的位置，变换后的View无点击效果
通用属性
android:fillAfter=”true” 设置动画停止后是否显示成原来的样子 false为恢复原样
android:duration=”300” 设置动画持续的时间
使用AnimationUtils.loadAnimation方法进行加载XML动画，使用startAnimator方法启动。

平移动画(translate)

平移动画的属性
主要是对坐标进行操作。
fromX/YDelta 原坐标 toX/YDelta 目的坐标

使用方式 - JAVA

使用方式 - XML

透明动画(alpha)

透明动画的属性
alpha 1表示不透明 0表示透明

java中加载方式同上。

缩放动画(scale)

理解
pivotX/Y 是指从哪个点开始缩放 50% 为中心原点 如果为0，则是左上角开始缩放> 到中间 一定要用百分数表示，用小数无效
fromX/YScale 开始的样子 默认1为原来的大小，0为(没有)，大于1为扩大。
toX/YScale 结束的样子 同上
示例：fromXY=”0” toXY=”1” 表示从没有到原来的样子进行缩放，fromXY=”1” > toXY=”1.5” 表示从原来的样子扩大1.5倍
fillAfter 结束动画是否保存 true-保存结束的样子 false-不保存
duration 设置动画时间 默认通常设置为300，单位毫秒

示例 - RecyclerView选中的Item变大

目的效果
使选中的Item放大,普通Item不变。
思路
根据中间偏移量是否为0决定是否缩放。
步骤
1.通过RecyclerView计算出中间位置的标准线。
2.计算Item的中间位置的线。(view.getLeft+view.getRight)/2
3.通过两者的差求偏移量。
4.调用setScale方法进行缩放。
注意
这里getLeft跟getRight都是坐标原点到left、right的水平距离。getRight不是view的右边框到最右边的距离。

旋转动画(rotate)

pivotX/Y 是指从哪个点开始旋转 50% 为中心原点 如果为0，则是左上角开始缩放到中间 一定要用百分数表示，用小数无效
repeatMode 是指重复模式 restart表示重复
repeatCount 指重复次数 -1(infinite)表示无限循环

属性动画(Property Animation)

理解
属性动画是补间动画的扩展。可以对对象及其属性进行操作，不限于View。
因为是对对象的属性的值进行修改，所以不会出现改变View位置后，点击View无效的情况。
属性动画有三种ValueAnimator、ObjectAnimator、AnimatorSet。
通过ofFloat、ofInt等方法来获取对象实例。
通过addUpdateListener来获取变化的值。

ValueAnimator

理解
控制值的变化。

ObjectAnimator

理解
对任意对象的任意属性进行动画操作。一次只能控制一个属性值(例如scaleX、要控制scaleY需要再获取一个对象)
ObjectAnimator.ofFloat(操作对象,操作的属性值,变化开始值,变化结束值)
操作的属性值可以是任意值，系统通过查找目标对象与该值相应的get、set方法来进行修改。当属性是自定义的时候，get、set方法的第一个字母必须是大写。例如mycolor对应getMycolor、setMycolor
通过addListener添加动画监听器，可以监听动画开始、结束等状态。

使用方式 - JAVA

使用方式 - XML

android:ordering=””：指定动画的播放顺序：sequentially(顺序执行)，together(同时执行)
android:propertyName=”属性名”：寻找该属性的get、set方法

TypeEvaluator

该接口用于设置动画系统如何从初始值过度到结束值
通过ofXXX方法传入对象，并且提供初始值，结束值
常用的构造方法

  • ofObject(使用属性的对象,属性值,计算器，初始值，结束值)
  • ofObject(计算器，初始值，结束值)
    ValueAnimator colorAnim = ObjectAnimator.ofObject(this, “mycolor”, new ColorEvaluator(), Color.BLUE, Color.RED);

Interpolator

该接口用于控制动画的变化速度
在xml动画中用到的interpolator属性，值类似驼峰命名法
android:interpolator=”@android:anim/linear_interpolator”

常用Interpolator 说明
AccelerateDecelerateInterpolator 在动画开始与结束的地方速率改变比较慢，在中间的时候加速
AccelerateInterpolator 在动画开始的地方速率改变比较慢，然后开始加速
AnticipateInterpolator 在动画初始位置向后甩一定的值
AnticipateOvershootInterpolator 在动画初始位置向后甩一定的值，在结束位置向后甩一定的值
BounceInterpolator 动画结束的时候弹起
CycleInterpolator 动画循环播 放特定的次数，速率改变沿着正弦曲线
DecelerateInterpolator 在动画开始的地方快然后慢
LinearInterpolator 动画匀速变化
OvershootInterpolator 回弹，超出结束位置然后回弹到结束位置

AnimatorSet

理解
动画集合
主要使用play、with、after、before等方法
