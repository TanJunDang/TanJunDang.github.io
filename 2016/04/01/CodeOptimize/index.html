ListView 选中优化

  • 点击选中代码优化

    • 原代码：

      convertView.setOnClickListener(new View.OnClickListener() {
    @Override
    public void onClick(View v) {
            User user = (User) v.getTag(dataTag);
            selectedUser = user;
            selectItem.clear();
            selectItem.add(selectedUser); //保存到集合中
            notifyDataSetChanged();
        }
});

.....

if (selectItem.contains(user)) { //通过集合判断是否包含该选项从而做相应的处理
    holder.ivCheck.setVisibility(View.VISIBLE);
} else {
    holder.ivCheck.setVisibility(View.GONE);
}

    • 优化后：

      convertView.setOnClickListener(new View.OnClickListener() {
    @Override
    public void onClick(View v) {
            User user = (User) v.getTag(dataTag);
            selectedUser = user; //直接用实体保存
            notifyDataSetChanged();
        }
});

......

holder.ivCheck.setVisibility(user.equals(selectedUser) ? View.VISIBLE : View.GONE);

  • 总结：集合方式比较适用于多选模式或者混合模式，实体模式适用于单选模式。
