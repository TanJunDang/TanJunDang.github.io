ListView 选中优化

  • 点击选中代码优化

    • 原代码：

      convertView.setOnClickListener(new View.OnClickListener() {
    @Override
    public void onClick(View v) {
            User user = (User) v.getTag(dataTag);
            selectedUser = user;
            selectItem.clear();
            selectItem.add(selectedUser); //保存到集合中
            notifyDataSetChanged();
        }
});

.....

if (selectItem.contains(user)) { //通过集合判断是否包含该选项从而做相应的处理
    holder.ivCheck.setVisibility(View.VISIBLE);
} else {
    holder.ivCheck.setVisibility(View.GONE);
}

    • 优化后：

      convertView.setOnClickListener(new View.OnClickListener() {
    @Override
    public void onClick(View v) {
            User user = (User) v.getTag(dataTag);
            selectedUser = user; //直接用实体保存
            notifyDataSetChanged();
        }
});

......

holder.ivCheck.setVisibility(user.equals(selectedUser) ? View.VISIBLE : View.GONE);

  • 总结：集合方式比较适用于多选模式或者混合模式，实体模式适用于单选模式。

相同样式使用共同布局

  • 目的要求

  • 思路

    • 数据跟布局结合组成可见的ItemView。
    • 数据：每个Item都由Title、Value、Arrow组成,所以新建一个类MsgItemBean，用于统一初始化数据。
    • 布局：每个Item都共用同一布局，所以可采用Holder的方式来FindId。

实例

  • 布局文件都是通过inclucde的方式加载同一布局,这里就不贴出布局代码了。

  • Bean类

    public class MsgItemBean {

    public static int ARROW_DISABLE = 0; //不显示箭头
    public static int ARROW_ENABLE = R.drawable.common_right_light_gray_arrow; //默认箭头
    private int titleResId; //标题的资源id
    private String value;//值
    private int arrowResId; //箭头
    private int hintResId; //值空的时候的提示

    public MsgItemBean(int titleResId, int hintResId, String value) {
        this(titleResId, hintResId, value, ARROW_ENABLE);
    }

    public MsgItemBean(int titleResId, int hintResId, String value, int arrowResId) {
        this.titleResId = titleResId;
        this.hintResId = hintResId;
        this.value = value;
        this.arrowResId = arrowResId;
    }
}
........//get、set方法

  • Holder类

    public class MsgHolder extends RecyclerView.ViewHolder {
    public View rootview;
    public TextView tvTitle;
    public TextView tvValue;
    public ImageView ivArrow;

    public MsgHolder(View itemView) {
        super(itemView);
        rootview = itemView;
        tvTitle = (TextView) rootview.findViewById(R.id.tvTitle);
        tvValue = (TextView) rootview.findViewById(R.id.tvValue);
        ivArrow = (ImageView) rootview.findViewById(R.id.ivArrow);
    }
}

  • 创建共用初始化方法

    //传入一个layout跟数据即可显示View
public void initItemView(ViewGroup layout, MsgItemBean data) {
    MsgHolder holder = new ProfileHolder(layout);
    holder.tvTitle.setText(data.getTitleResId());
    holder.tvValue.setHint(data.getHintResId());
    holder.tvValue.setText(data.getValue());
    if (data.getArrowResId() != 0) {
        holder.ivArrow.setVisibility(View.VISIBLE);
        holder.ivArrow.setImageResource(R.drawable.common_right_light_gray_arrow);
    } else {
        holder.ivArrow.setVisibility(View.GONE);
    }
}

  • 实际使用

    initItemView(rlPhone, new ProfileItemBean(R.string.common_text_phone, R.string.common_text_phone, “13631450006”, ProfileItemBean.ARROW_DISABLE));
