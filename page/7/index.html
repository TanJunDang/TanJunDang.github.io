PopupWindow笔记

PopupWindow 基本用法

总结
　PopupWindos的作用就是设置显示View的宽高、背景、动画以及显示的位置。

步骤
1.用LayoutInflate加载待显示的布局。
2.创建PopupWindow对象，传入上述view以及宽高作为参数。(PopupWindow(View contentView, int width, int height，boolean focusable))//focusable表示点击外部是否可以关闭
3.设置PopupWindow的背景。(setBackgroundDrawable)
4.设置PopupWindow的显示位置，并且将其显示。(两种方法)
5.popupWindow.setFocusable(true);//设置popupwindow获取焦点
6.popupWindow.setOutsideTouchable(true); // 设置允许在外点击消失

开发中常见的Error、Bug

常见的错误

ToolBar设置透明色无效

原因
ToolBar的父布局为AppBarLayout，AppBarLayout没有设置背景色，所以默认会调用style中的colorPrimary。因此透明色无效

解决方法
1.设置AppBarLayout的背景色。
2.修改style里colorPrimary的颜色。

自定义Button时,无法响应UP/MOVE事件

需要重写isClickable方法，使其返回true。

Using incompatible plugins for the annotation

Warning:Using incompatible plugins for the annotation processing: android-apt. This may result in an unexpected behavior.

解决方法

Gradle sync failed: Connection refused: connect

打开gradle.properties,把下列代码注释掉

Build出错the android.dexOptions.incremental

下列代码中删除incremental true

找不到binding

重启AS
可能是某个类删除了某个import

找不到R文件

build之后突然找不到R文件
原因
最近改的布局文件里缺少了一些资源文件,例如缺少了drawable文件ic_load_image，布局里调用了@drawable/ic_load_image。

复制布局文件时，有自定义view，要注意改其包名。
