ListView 分割线

在divider 为颜色 时 需设置dividerHeight
如何判断listview是最后一个

if(坐标<总长度-1)

如何取消分割线

  • 在java中取消 

    listView.setDividerHeight(0);

listView.setDivider(null);

  • 在XML布局文件取消 

    android:divider="#FFCC00"  

或      

android:divider="@null"

设置listview无点击背景效果

android:listSelector="@android:color/transparent"

ConvertView的理解

  • 每一个条目（Item）的显示，都是ListView加载布局来实现的。
  • convertview就是滑出屏幕后被Android回收的条目。

原因：由于每次加载布局会使性能下降，所以应该复用convertview。

如何复用？ 代码如下：

if(convertView==null){
    加载布局
}
//然后获取布局里的的View设置相关的值  
convertView.findViewById(R.id.text)).setText(DATA[position]);

由于每次都要FindId，这样也会使性能下降。所以我们应该复用ConvertView，这时就用到ViewHolder跟settag跟gettag方法了。

  • 简单来说ViewHolder就是对一个布局里的控件FindId

所以代码改成这样： 

ViewHolder holder;
if(convertView==null){
    加载布局
holder =  new ViewHolder();  
convertView.setTag(holder);
}else{
holder = (ViewHolder)convertView.getTag();
}
//然后获取布局里的的View设置相关的值  
holder.tvName.setText(DATA[position]);
.
.

class ViewHolder{
    private TextView tvName;
}

ListView的下拉刷新，上拉加载数据

在跟服务器交互时

  • 当ListView 下拉时，清空原数据,请求新的一页数据(用currentPage判断)。 currentPage = 1 ;

  • 当Listview 上拉时，重新加载一页数据。 currentPage ++;
