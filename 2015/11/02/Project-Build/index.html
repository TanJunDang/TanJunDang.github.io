项目架构基本目录

思路 项目架构基本分为三种类别通用模块包、业务模块包、全局基础包。

  • 通用模块包（common） 包括工具包、实体包、控件包、网络包、数据库包等

    • Broadcast(带接口的广播、EventBus事件)

    • Dao(数据库)

    • Entities(实体类)

    • Utils

      • 文件操作类封装
      • 基础工具类Function的编写（包括Toast封装、SD操作等基础功能）
      • Log封装（日志类的管理）
      • Gson封装
      • ImageLoader封装
      • SharePreference封装

    • Model (用于缓存数据)

    • Network(protocol协议包、各种网络请求接口、CallBack)

    • View(各种自定义View)

  • 业务模块包（以业务命名） 每个功能模块单独成包
  • 全局基础包（base） 做Application初始化相关操作、异常捕获类。
