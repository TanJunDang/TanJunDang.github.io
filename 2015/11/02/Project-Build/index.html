发表于 2015-11-02   |  

项目架构基本目录

  • 通用包common 包括工具包、实体包、控件包、网络包、数据库包等

    • Broadcast(带接口的广播、EventBus事件)

    • Dao(数据库)

    • Entities(实体类)

    • Utils

      • 文件操作类
      • Toast封装
      • Log封装
      • Gson封装
      • ImageLoader封装
      • SharePreference封装
      • SD卡相关类封装

    • Model (缓存数据)

    • Network(protocol协议包、各种网络请求接口、CallBack)

    • View(各种自定义View)

  • 模块包(每个功能模块单独成包)
  • 全局基础包base(Application初始化相、异常捕获类)

Utils包的基础类

  • 日志管理类LogUtil:主要用于过滤日志和控制日志的开关。

思路：打包时可以控制Log的开关，可以将Log输出到文件。
使用：将日志写入到文件的基本步骤是先找到路径，再创建文件，再写入日志到文件。
