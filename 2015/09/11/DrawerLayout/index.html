总结

ActionBarDrawerToggle作用

1.改变返回图标。
2.Drawer拉出、隐藏，动画效果。
3.监听Drawer拉出、隐藏；

一般情况下DrawerLayout跟ActionBarDrawerToggle结合使用。

ActionBar相关方法

getSupportActionBar().setDisplayHomeAsUpEnabled(false); // 返回箭头（默认不显示）
getSupportActionBar().setDisplayShowHomeEnabled(false); // 是否显示左上角图标(系统)
getSupportActionBar().setDisplayShowTitleEnabled(false); // 是否显示标题
getSupportActionBar().setDisplayShowCustomEnabled(true); //是否显示自定义视图

注意

1.布局顺序不能变。Android系统解析顺序为主布局、左抽屉、右抽屉。
2.重写onPostCreate、onConfigurationChanged方法，用于屏幕旋转时，同步状态。
3.重写onOptionsItemSelected方法解决点击Toogle无效的问题。
4.如果要使用ToolBar，则Activity要设置成无ActionBar主题，调用ActionBarDrawerToggle有ToolBar参数的构造函数。
