Android网络层框架设计

理解
网络层框架设计分以下部分
定义请求实体基类、定义请求协议基类、封装请求管理类、定义业务回调接口

定义请求实体基类

该类主要定义错误码、错误信息、状态码等

定义请求协议基类

该类主要封装请求参数，拼接、加密URL并返回。

定义请求实体基类

定义请求管理类(RequestManager)

该类主要提供同步、异步请求url的方法以及上传下载的方法。

定义业务回调接口(RequestCallback)

该类主要是提供请求业务的回调接口以及上传下载的回调接口。
